# ZImage Icons

This directory contains icons used throughout the ZImage Enterprise application.

## Required Icons

The application requires the following icons:

- `folder.png` - For opening folders
- `file.png` - For opening files
- `edit.png` - For editing images
- `resize.png` - For resizing images
- `back.png` - For navigation back
- `up.png` - For navigation up
- `home.png` - For navigation home
- `refresh.png` - For refreshing the current view
- `new.png` - For creating new images
- `open.png` - For opening images
- `save.png` - For saving images
- `save_as.png` - For saving images with a new name
- `undo.png` - For undoing actions
- `redo.png` - For redoing actions
- `reset.png` - For resetting to original image
- `pen.png` - For pen tool
- `line.png` - For line tool
- `rectangle.png` - For rectangle tool
- `ellipse.png` - For ellipse tool
- `text.png` - For text tool
- `presets.png` - For resize presets

## Icon Format

Icons should be PNG format with transparency, preferably 24x24 pixels for toolbar icons.
